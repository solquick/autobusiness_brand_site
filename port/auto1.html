프로그램 간단설명

특정 파일에 저장된 문구들을 자동으로 번역하고 원하는 양식으로 출력해줍니다.
구글, chag gpt 등의 번역을 위한 api를 이용해서 빠르게 번역을 진행합니다.
영상제작, 자막 생성 등의 업무에 용이합니다.

파일에 문장 저장

엑셀파일(또는 다른 파일)에 번역해 필요한 문장들을 입력합니다.
여러 파일에 나누어서 저장해도 괜찮습니다.

프로그램 실행

프로그램을 켜고 파일찾기 버튼을 누릅니다.
번역을 하고 싶은 파일 혹은 그 파일이 모여있는 폴더를 선택합니다.

번역 실행

추출시작 버튼을 눌러서 대량의 번역을 자동으로 진행합니다.
번역은 chat gpt의 api 혹은 google 번역 api를 사용해서 번역합니다.
번역된 결과물은 원하는 형식의 파일에 저장됩니다.

엑파일에 저장된 결과물

추출이 끝나면, 결과물들이 (엑셀)파일에 모두 저장됩니다.
번역된 결과물들이 원하는 형식의 파일에 저장됩니다.
이 결과물은 번역자막, 영상편집 등 다양하게 사용될 수 있습니다.
