나라장터 크롤링 프로그램 메인화면

나라장터 크롤링 프로그램의 메인 화면입니다.
입찰공고조사 버튼을 누르면, 입찰공고의 데이터 추출이 시작되고,
발주계획조사 버튼을 누르면, 발주공고의 데이터 추출이 시작됩니다.
저장 버튼을 누르면, 추출된 데이터가 엑셀파일로 저장됩니다.

키워드 메모장에 입력

키워드는 메모장에 입력합니다.
하나의 키워드씩 입력하고, 각 키워드는 엔터로 구분합니다.
메인프로그램이 있는 폴더와 같은 폴더에 keyword.txt 라는 이름으로 저장해줍니다.

엑셀파일에 저장된 결과물

저장 버튼을 누르시면, 추출된 데이터가 엑셀파일로 저장됩니다.
결과물 엑셀파일은 메인프로그램이 있는 폴더와 같은 위치에 생성됩니다.
정돈된 결과물을 한 눈에 볼 수 있고, 추후에 원하신다면 데이터를 이용한 분석도 가능합니다.
