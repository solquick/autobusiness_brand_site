일본 쇼핑몰 크롤링 프로그램

프로그램 간단설명

일본(해외) 쇼핑몰 사이트의 판매데이터를 자동으로 수집하는 프로그램입니다.
수백개의 조사하고 싶은 품목코드로 검색된 상품의 이름, 링크, 가격, 배송가(옵션별)를 자동으로 추출하고 저장합니다.

프로그램 메인화면

일본 쇼핑몰 크롤링 프로그램의 메인 화면입니다.
추출 시작 버튼을 누르면 추출이 시작됩니다.
추출이 진행되는 동안, 현재 어디까지 추출이 되었는지 로그기록이 남습니다.

양식파일에 부품번호 입력

추출을 원하는 상품의 부품번호들을 엑셀파일에 입력합니다.
해당 부품번호들을 모두 조회하면서 해당 상품의 정보를 자동으로 추출합니다.
양식은 자유롭게 설정이 가능하고, 수식도 설정 가능합니다.

엑셀파일에 저장된 결과물

추출이 끝나면, 엑셀파일에 추출결과물이 자동으로 저장됩니다.
노란색 셀들이 추출된 결과물들입니다.
양식파일에 수식이 적혀있었다면, 추출된 결과물과 함께 연산이 됩니다.

여러분의 업무에 자동화의 날개를 달아드리겠습니다.
